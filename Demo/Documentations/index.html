Getting started

Previously, I thank you because you have to use an application that we developed.

Add Folder To List

You can do it by follow this step:

  1. First, you must have already installed this app
  2. In an application, press the button Add Folder. to add a new folder to the list
  3. Then browse the folders you want to be locked, and press the Add button as shown below

    1. Click to browse the folder, or you can type the destination in textbox
    2. Then click Add to add new folder to the list

  4. Folders that had been added will automatically appear in the list

Remove Folder From The List

Previously, you had to have a folder in the list (at least one folder)

  1. Click the folder you want to Delete from the list
  2. Then press the Remove button

Lock Folder

Previously, you had to have a folder in the list (at least one folder) with a Locked status

  1. Click the folder you want to Lock from the list
  2. Then press the Lock button

Unlock Folder

Previously, you had to have a folder in the list (at least one folder) with a Unlocked status

  1. Click the folder you want to Lock from the list
  2. Then press the Lock button

Uninstalling The Program

Before you uninstall this app, please Unlock all folders that have been Locked
